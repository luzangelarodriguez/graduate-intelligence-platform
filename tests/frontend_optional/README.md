# Frontend Optional Tests

Reservado para pruebas visuales o Playwright del frontend React.

Esta fase no modifica frontend ni exige pruebas E2E visuales. Cuando el backend y la calidad de datos pasen, esta carpeta puede incorporar:

- smoke test de login;
- navegacion del observatorio;
- cambio de programa activo;
- render responsive;
- ausencia de errores de consola.
